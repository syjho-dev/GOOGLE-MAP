# StoryMaker Clone (Google Map Style Photo Frame Generator)

이 프로젝트는 사진을 업로드하면  
장소명 · 별점 · 리뷰 · 카테고리 텍스트를 자동으로 합성하여  
**스토리메이커(Storymaker) 스타일의 프레임 이미지**를 생성하는 웹 서비스입니다.

기술 스택은 아래와 같습니다:

- **Next.js 14 (App Router)**
- **TailwindCSS**
- **Sharp (이미지 합성)**
- **Vercel (무료 배포)**
